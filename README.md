# muhammad.lol

Personal portfolio site built with Astro, TypeScript, and Tailwind CSS.

Live at [muhammad.lol](https://muhammad.lol)

## Stack

- [Astro](https://astro.build) — static site framework
- TypeScript
- Tailwind CSS
- Framer Motion — animations

## Running locally
```bash
npm install
npm run dev
```

## Deploying

Automatically deploys to GitHub Pages via GitHub Actions on every push to `main`.
